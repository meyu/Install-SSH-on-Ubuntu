Install-ssh-on-ubuntu

於 Ubuntu 上安裝 Secure Shell (SSH)

Download this project as a .zip file Download this project as a tar.gz file

所為何來

於 Ubuntu 上安裝 SSH 協定，以使用加密的遠端存取協定，來進行遠端操作、檔案傳輸。

使用環境

Ubuntu Server 12.04

安裝方式

於終端機中，輸入以下指令即可：

sudo aptitude install ssh

DONE.
