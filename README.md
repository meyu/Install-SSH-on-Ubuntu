所為何來
=
於 Ubuntu 上安裝 SSH 協定，以使用遠端連線。

使用環境
=
Ubuntu Server 12.04

安裝方式
=
於終端機中，輸入以下指令即可：
```bash
sudo aptitude install ssh
```

<br>
<br>

補充說明
=
###Secure SHell (SSH)
>為一種加密的遠端存取協定，可用於遠端操作、檔案傳輸。  
>Ubuntu 預設已內建 OpenSSH Client。
